# Express
## node.js安装
* 安装node.js <br>
  sudo npm install -g node<br>
* 验证是否安装成功 <br>
  node -v <br>

## express安装
* 先安装express-generator <br>
  npm install -g express-generator <br>
* 安装express <br>
  npm install -g express <br>
* 验证是否安装成功 <br>
  express --version <br>

## 新建项目
*  创建模版 <br>
   express 项目名 <br>
*  切换到项目目录下 <br>
   cd 项目名 <br>
*  安装项目依赖模块 <br>
   npm install <br>

## 启动项目
   npm start <br>

## 浏览器查看
   http://localhost:3000 <br>
